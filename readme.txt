# Posts Coverflow Slider

**Description:** Posts Coverflow Slider is a WordPress plugin that allows you to create a beautiful coverflow slider for your posts, enhancing the visual appeal of your website. Easily showcase your latest posts in an engaging and dynamic way.

**Shortcode Examples:**
- Display all posts with the default settings:
  `[latest_posts_slider]`

- Display 5 latest posts from a specific category:
  `[latest_posts_slider posts_per_page="5" category="your_category_slug"]`

## Installation

1. Upload the `posts-coverflow-slider` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

## How to Use

Once the plugin is activated, you can use the provided shortcode to embed the coverflow slider into your posts, pages, or widgets.

- Add the following shortcode to display the coverflow slider:
  `[latest_posts_slider]`

- Customize the shortcode attributes as needed:
  - `posts_per_page`: Number of posts to display.
  - `category`: Slug of the category to include.

Feel free to experiment with different shortcode attributes to suit your preferences and website design.
